# 即刻&小宇宙的todolist
## 后端
后端采用gozero搭建微服务框架，启动流程是先去task和user目录下分别进入cmd/xxx/rpc启动rpc服务，然后再去网关层的api下启动网关服务，前提是本地已安装mysql,redis,etcd，需要去yaml配置文件修改数据库连接地址
### 启动流程
cd backend
go mod tidy
然后执行上述服务user，task，gateway

可以通过打包build或者docker启动，但是我就在开发环境完成了
## 前端
前端采用vue3组合式api+ts，大部分通过ai生成，使用pnpm作为包管理工具
### 启动流程
cd frontend
pnpm install
pnpm dev
